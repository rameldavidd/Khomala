# Khomala Website — Reorganized Sections

This version separates website photos into section folders so the same gallery photos are not repeated everywhere.

## Main structure

- `assets/images/hero/` — homepage hero photos
- `assets/images/story/` — Aylin, Basim, and family story photos
- `assets/images/featured/` — featured looks shown near the top
- `assets/images/families/` — family and children section photos
- `assets/images/details/` — custom design and detail photos
- `assets/images/orders/` — custom order section photo
- `assets/images/community/` — bottom community shared gallery only

## Important change

The bottom gallery is now only **Community Shared Photos**. Featured looks, family sets, and design details are shown in their own sections above.

## Hosting

Upload the contents of this folder to GitHub Pages, with `index.html` at the top level of the repository.
